# Food-Delivery-App-Analysis-PowerBi

- Project Overview

The OMATO Food Delivery Dashboard is an interactive Power BI report designed to analyze food delivery performance, customer transactions, and sales trends. This dashboard provides clear insights into total quantity sold, transaction volume, payment methods, delivery performance, and monthly trends.

It helps business stakeholders make data-driven decisions by visualizing key metrics in a simple and structured format.

- Key Performance Indicators (KPIs)

Total Quantity Sold: 15K
Total Transactions: 2.75K
Average Quantity per Transaction: 5.50

These KPIs provide a quick summary of overall business performance.
